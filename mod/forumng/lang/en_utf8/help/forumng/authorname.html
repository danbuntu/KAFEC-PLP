Author name

You can type part or all of an author's real name (the 'real name' is the name that displays next to their posts) into this box, case-insensitively.

For example:

if you typed chris you could match posts by Christian, Christianne, Christine, Chris, Chrissie, Christopher, etc.

if you typed Woo you could match posts by Wood, Woolhead, Woodhead, Woodcock, Woodley, etc.

if you type a complete forename John and initial B you will match posts by John Barnet, John Beedell, John Berriman, etc.

Alternatively, you can type a username which must be complete.

For example:

if you typed cw1234 you could match posts by the author with this username.
