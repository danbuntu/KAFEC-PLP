Move discussions to

Choose what you want to do with removed discussions.

Delete permanently

The default option Delete permanently will delete these discussions.

Ordinarily, when a moderator deletes a post, the post is not 'really' deleted; it can be restored. This is not the case for automatically deleted discussions, which are permanently deleted and cannot be recovered.

Move to another forum

Alternatively, you can select a forum in this course to move these discussions into. For example, some teachers like to create a hidden 'Archive forum' which is used to store old discussions.

You can only move discussions to a forum with compatible group settings.

  • Discussions can always be moved into a forum set to No groups.
  • If the target forum is set to Separate groups or Visible groups, then it must have the same grouping as the source (current) forum.
