Automatically / manually marking discussions read

Click the Change option to toggle between automatically, or manually, marking discussions as read.

When you have not yet read posts, they display with a different background colour and other effects so that you can tell they are unread. This setting controls how such posts are marked read, losing their special appearance.

  • By default, discussions are automatically marked read whenever you view them. If you view a discussion containing unread posts, the system assumes you are going to read those posts, so it marks them read. The next time you view the discussion, they will be shown as normal read posts.
  • If you like, you can change it so that you manually mark discussions read. When you finish reading a discussion, click the Mark discussion read button at the bottom of the discussion page. You can also use the tick mark icons or the Mark all posts read button on the discussion list.

The automatic/manual option affects all forums.
