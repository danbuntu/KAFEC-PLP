Individueller Text

Wenn Sie auf dem Zertifikat anderen Trainern/innen namentlich ausdrucken wollen als Sie in den Kursrollen festgelegt haben, schalten Sie den Ausdruck des Trainernamens oder der Signatur/Unterschrift ab. Geben Sie in der Textbox den Titel ein, wie er gedruckt werden soll. In der Grundeinstellung wird die Eingabe unten links gedruckt. Sie können dies in der Datei certificate/type/"type name"/certificate.php anpassen.

In der Datei finden Sie ziemlich weit unten folgende Codezeile:

cert_printtext(150, 450, '', '', '', '', '');

Die beiden Zahlen beschreiben die horizontale und vertikale Platzierung ausgehend von der linken oberen Ecke. 'X' beschreibt den horizontalen, 'Y' den vertikalen Wert. Passen Sie die Angaben entsprechend Ihren Wünschen an.

In der Textbox können Sie auch direkt HTML-Code eingeben, z.B. einen Link oder die URL zu einem Bild.

Folgende Html-Tags können verwendet werden:
  • <br> und <p>
  • <b>, <i> und <u>
  • <img> (src und width (oder height) sind erforderlich)
  • <a> (href ist erforderlich)
  • <font>: optionale Attribute sind:
    color: hex color code
    face: arial, times, courier, helvetica, symbol

Beispiel Html-Text:

Mr. James Salesman, Manager<br><br>Verkaufsabteilung<br><br><font color="#0000CC"><b>Ihre Firma<font face="Symbol">&Ograve;</font></b></font><img src="http://yourmoodle.com/mod/certificate/pix/seals/Logo.png" width="100"><p><a href="http://www.site.com target="_blank">Klick hier</a></p>
